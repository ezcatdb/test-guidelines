Enzyme names including abbreviated names are annotated.

Moreover, general description of enzymes should be annotated as follows:
- enzyme(s)
  - both enzymes
  - each enzyme
  - either enzyme
  - homologous enzyme(s)
  - key enzyme(s)
- mutant enzyme(s)
  - E128H (mutant) enzyme
  - E198A (mutant) enzyme
  - E198Q (mutant) enzyme
- wild-type enzyme
- native enzyme
